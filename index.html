Pick a video from Google Drive, mark the useful section, add metadata, and save a clean copy. Everything runs in this browser tab — no application server or separate upload.

FrameCut uses Drive access—not Gmail—to browse and download the video you choose, then save or share your trimmed copy. The short-lived token stays in this tab and is revoked when you sign out.

How to use this
  1. Open a folder to browse; click a video to load it.
  2. Drag the timeline handles (or press I/O) to set the start and end.
  3. Add any metadata, then Trim and save to Drive — the trimmed copy lands in the folder you choose.
  4. Tick anyone with the link can view to get a shareable link.
